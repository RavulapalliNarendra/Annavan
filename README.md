🌾 Annavan – Farmer to Market Platform

📌 Overview

Annavan is a digital agriculture platform designed to connect farmers directly with buyers, eliminating middlemen and ensuring fair market prices. The platform empowers farmers to sell their crops efficiently while providing buyers with access to fresh agricultural products directly from the source.

Our mission is to create a transparent, technology-driven marketplace that improves farmer income and strengthens the agricultural supply chain.

---

🚀 Features

👨‍🌾 Farmer Module

- Farmer Registration & Login
- Crop Listing Management
- Product Upload with Images
- Market Price Visibility
- Order Notifications
- Direct Buyer Communication

🛒 Buyer Module

- Buyer Registration & Login
- Browse Available Crops
- Search & Filter Products
- Place Orders Directly
- Track Purchase Requests

📊 Smart Features

- Real-Time Market Updates
- Direct Farmer-to-Buyer Transactions
- Transparent Pricing
- Crop Availability Dashboard
- Secure Authentication

💡 Problem Statement

Farmers often receive lower prices due to multiple intermediaries in the agricultural supply chain. Limited access to market information and direct customers reduces farmer profitability.

Annavan solves this problem by creating a direct marketplace where farmers and buyers can connect without middlemen.

---

🎯 Objectives

- Increase farmer profits

⚙️ Installation

Clone Repository

git clone https://github.com/YOUR_USERNAME/annavan.git
cd annavan

Install Frontend Dependencies

npm install

Install Backend Dependencies

pip install -r requirements.txt

Run Frontend

npm start

Run Backend

uvicorn app:app --reload
